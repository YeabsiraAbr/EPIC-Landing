# Epic Software Solutions - Company Profile

A professional company profile website built with Next.js 15, showcasing Epic Software Solutions' services, team, and contact information.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```

## Features

- Modern, responsive design
- White and black color theme
- Professional company profile sections
- Contact information
- Services showcase
- About Us section
